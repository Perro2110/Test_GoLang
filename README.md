## 📖 About

This repository contains my initial Go programs, written while following **[Go by Example](https://gobyexample.com)** — a hands-on introduction to Go through annotated example programs.

Each file is a self-contained example focused on a specific language feature or concept.

---

## 🚀 Running the Programs

```sh
# Compile and run directly
go run [filename.go]

# Compile and generate a binary
go build [filename.go]
```
